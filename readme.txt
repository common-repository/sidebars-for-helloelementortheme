Sidebars for Hello Elementor Theme is a WordPress plugin adds sidebars support for Hello Elementor WordPress theme.

== Description ==

This plugin comes with 6 responsive sidebars for just about everything you need to use anywhere within your website built using Elementor and Hello Elementor theme.

<a href="https://wordpress.org/plugins/elementor/" rel="nofollow">Elementor</a> plugin must be activated to use this plugin. After you activate the required plugins, the sidebars should be available for use in Elementor.


<strong>Sidebars</strong>:

<ul>
<li>General sidebar</li>
<li>Blog sidebar</li>
<li>Footer 1</li>
<li>Footer 2</li>
<li>Footer 3</li>
<li>Footer 4</li>
</ul>

<strong>Sidebars for Hello Elementor Theme</strong> features allows you to:

<ul>
<li>Set background for Sidebars</li>
<li>Control Sidebars appearance using Elementor</li>
<li>Live update using WordPress Customize</li>
</ul>

== Installation ==

1. Install and activate the Elementor page builder.
1. Install and activate the Hello Elementor theme.
2. Unzip the downloaded sidebars-for-hello-elementor-theme.zip file and upload to the `/wp-content/plugins/` directory or install the Sidebars for Hello Elementor Theme by Mnbaa plugin from WordPress repository. Activate the plugin through the 'Plugins' menu in WordPress.
4. Add "Sidebar" Element using Elementor page builder to your templates, pages or theme pages.
5. Choose your desired sidebar from the dropdown list for sidebars.

== Frequently Asked Questions ==

= Does it work with the theme that I am using? =

My tests indicate that sidebars work well with most themes that are well coded. You may need some minor custom CSS with themes that hijack the styling.
